# IOSSDK-Example
## setup
### 1. Install the library
```
pod install
```

**_Note:_** if not working try to :
```
pod deintegrate
pod clean
pod setup
pod install
```

## UrlSession
By default this Example App use UrlSession

## Alamofire

To use alamofire you need to:
- uncoment all Alamofire section 
- comment all UrlSession section
